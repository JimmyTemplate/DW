What are the materials

The usage of material differentiate in each college or school that implemented the technique of card game but a common material is page. Page is a term referred to multiple combination of color that combined to create a picture in which the page also has a different resolution such as 1:25 or 1:16. Not only the page but the other material such as luminous paper might be use bu some school to create a light code for example.
